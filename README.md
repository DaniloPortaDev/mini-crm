https://daniloportadev.github.io/mini-crm/

---

## ✨ Funcionalidades

- Cadastro de clientes
- Edição de informações
- Remoção de registros
- Busca por nome ou e-mail
- Persistência de dados com localStorage
- Interface responsiva

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- localStorage

---

## 📊 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais do desenvolvimento web, incluindo:

- Manipulação de DOM
- Eventos em JavaScript
- Estruturação de dados no front-end
- Persistência de dados no navegador

A aplicação simula um sistema básico de CRM (Customer Relationship Management), permitindo gerenciar clientes de forma simples e eficiente.

---

## 🎯 Objetivo

Consolidar conhecimentos em JavaScript puro (Vanilla JS) e evoluir na construção de aplicações web interativas sem o uso de frameworks.

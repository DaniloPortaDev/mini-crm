# Mini CRM

Aplicacao web simples para cadastro e organizacao de clientes, com persistencia local no navegador.

## Tecnologias
- HTML
- CSS
- JavaScript
- localStorage

## Funcionalidades
- Adicionar clientes
- Editar clientes cadastrados
- Remover clientes
- Buscar clientes por nome ou email
- Interface responsiva e mais organizada para apresentacao
- Persistencia de dados com localStorage

## Deploy
[Acessar projeto](https://daniloportadev.github.io/mini-crm/)

## Sobre o projeto
Este projeto foi desenvolvido como pratica de manipulacao de DOM, eventos e armazenamento local em JavaScript.

## Uso de IA
Este resultado contou com apoio de IA no processo de refinamento da interface, organizacao da experiencia de uso e melhoria da apresentacao final do projeto.
